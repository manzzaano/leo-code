# kc-core

Librería base compartida del sistema KC-RAG.

Parte del sistema [leo-code](https://github.com/manzzaano/leo-code).

## Contenido

- **Capsule** — unidad de código indexada (función/clase/módulo con metadatos)
- **Parser** — extracción AST de Python sin LLM
- **serialize_context** — serialización de subgrafos a texto comprimido

## Instalación

```bash
pip install kc-core
```
